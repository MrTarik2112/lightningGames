## 🎯 FPS Counter

### How to Enable:
1. Open Settings (⚙️)
2. Visual Effects section
3. Check "Show FPS Counter"

### Color Codes:
- **Green** (55+ FPS) - Excellent performance
- **Yellow** (30-54 FPS) - Acceptable performance
- **Red** (<30 FPS) - Poor performance

### Tips for Better FPS:
1. Lower Render Scale to 70-80%
2. Reduce Particle Density to 50%
3. Lower Glow Intensity to 50%
4. Enable Reduced Motion
5. Close other applications

## 💡 Tips & Tricks

### Achievement Hunting:
1. Play different games to unlock Explorer
2. Play same game repeatedly for Persistent
3. Play at different times for time-based achievements
4. Try all themes for Theme Collector
5. Mark favorites for Favorite Collector

### Performance Optimization:
1. Lower Render Scale for better FPS
2. Reduce Particle Density on slower PCs
3. Enable Reduced Motion for minimal animations
4. Lower Glow Intensity to reduce GPU load

### Customization:
1. Try all 11 themes to find your favorite
2. Adjust Card Size for your screen
3. Enable Compact Mode for more games visible
4. Customize audio levels independently

### Gameplay:
1. Enable Auto-Pause if you multitask
2. Enable Confirm Exit to prevent accidents
3. Show Timer to track your play sessions
4. Adjust Difficulty for challenge level

## 🔧 Troubleshooting

### Low FPS:
- Lower Render Scale to 70%
- Reduce Particle Density to 0%
- Lower Glow Intensity to 50%
- Enable Reduced Motion
- Close background applications

### Audio Issues:
- Check SFX Volume and Music Volume
- Disable Mute on Blur if needed
- Restart the application

### Visual Issues:
- Try different themes
- Adjust Glow Intensity
- Reset settings to default
- Restart the application

### Achievement Not Unlocking:
- Check if already unlocked in settings
- Ensure you meet exact requirements
- Some achievements are hidden
- Restart game and try again
