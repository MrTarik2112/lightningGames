# Settings Menu v2.0 - Quick Start Guide

**Get started with the Enhanced Settings Menu in 5 minutes**

---

## 🚀 Quick Start (5 Minutes)

### Step 1: Open Settings (30 seconds)
1. Click the **⚙️ Settings** button in the launcher
2. Settings panel appears with fade-in animation
3. Display tab is active by default

### Step 2: Explore Tabs (1 minute)
Click each tab to explore:
- **🎨 Display** - Themes, visual effects
- **🔊 Audio** - Volume controls
- **🎮 Gameplay** - Difficulty, pause options
- **📱 Interface** - Layout customization
- **💾 Data** - Backup, export, import

### Step 3: Change Theme (1 minute)
1. Go to **Display** tab
2. Click any theme button (11 options)
3. Theme applies instantly
4. Selection is saved automatically

### Step 4: Adjust Volume (1 minute)
1. Go to **Audio** tab
2. Drag **Sound Effects Volume** slider
3. Drag **Music Volume** slider
4. Changes apply in real-time

### Step 5: Close Settings (30 seconds)
1. Click **✕** button in top-right
2. Or press **Escape** key
3. Settings are saved automatically

## 🎚️ Slider Controls

### How to Use Sliders
1. **Click** on slider to set value
2. **Drag** thumb to adjust
3. **Arrow keys** to fine-tune (when focused)
4. **Value** updates in real-time

### Available Sliders
- Screen shake intensity (0-100%)
- Resolution scale (70-100%)
- Particle density (0-100%)
- Glow intensity (0-100%)
- Animation speed (50-200%)
- SFX volume (0-100%)
- Music volume (0-100%)
- Card size (80-120%)

## 💾 Data Management

### Export Data (Backup)
```
1. Go to Data tab
2. Click "Export All Data"
3. JSON file downloads automatically
4. File includes: scores, achievements, settings
5. Filename: lightning-games-backup-{timestamp}.json
```

### Import Data (Restore)
```
1. Go to Data tab
2. Click "Import Data"
3. Select previously exported JSON file
4. Review data preview
5. Click "Import" to confirm
6. Page reloads with new data
```

### Hard Reset (Delete All)
```
1. Go to Data tab
2. Scroll to "Danger Zone"
3. Click "Hard Reset"
4. Confirm twice (safety)
5. All data deleted permanently
```

## 🐛 Quick Troubleshooting

### Settings Not Saving
- Check if localStorage is enabled
- Try clearing browser cache
- Restart the application

### Theme Not Applying
- Reload the page
- Clear browser cache
- Try different theme

### Data Import Fails
- Verify JSON file is valid
- Check file is from Lightning Games
- Try re-exporting data

### Export File Not Downloading
- Check browser download settings
- Disable popup blocker
- Try different browser

## 💡 Pro Tips

### Performance
- Lower resolution scale for better FPS
- Reduce particle density for smoothness
- Disable animations for minimal motion

### Audio
- Set SFX to 100% for full effects
- Set music to 50% for ambiance
- Enable "Mute on Blur" to save battery

### Gameplay
- Enable "Auto-Pause" for focus loss
- Enable "Confirm Exit" to prevent accidents
- Set difficulty to "Hard" for challenge

### Interface
- Enable "Compact Mode" for more games
- Increase "Card Size" to 120% for easier clicking
- Set "Grid Layout" to 6 for overview

### Data
- Export data weekly for backup
- Store backups in safe location
- Test import before deleting data
