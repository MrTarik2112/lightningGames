## 🔧 Common Commands

```bash
# Standard build (interactive)
npm run dist

# Windows only
npm run build:win

# Linux only
npm run build:linux

# All platforms
npm run build:all

# Clear cache
node scripts/build.js --clear-cache

# Show help
node scripts/build.js --help
```
