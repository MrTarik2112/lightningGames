# ⚙️ Settings

All setting options to customize your Lightning Games experience.

---

## 📊 Tab Structure

Settings are organized into 5 main tabs:

| Tab | Icon | Content |
|-----|------|---------|
| **Display** | 🎨 | Themes, visual effects, resolution |
| **Audio** | 🔊 | Volume controls |
| **Gameplay** | 🎮 | Difficulty, pause, timer |
| **Interface** | 📱 | Layout, card size, grid |
| **Data** | 💾 | Export, import, reset |

---

## 💡 Tips

1. **FPS counter** - Open if you have performance issues
2. **Reduced motion** - Use in low power mode
3. **Compact mode** - To see more games at once
