# 🚀 Getting Started

Follow this guide to start using Lightning Games.

---

## 📥 Installation

### 1. Install Dependencies

```bash
# With npm
npm install

# Or with Bun (3-5x faster)
bun install
```

### 2. Launch the App

```bash
# With npm
npm start

# Or with Bun
bun start
```

---

## 🎮 Opening the App

Once the app is running, you can open it in several ways:

| Method | Description |
|--------|-------------|
| **Hotkey** | `Ctrl+Alt+G` - Opens from anywhere |
| **System Tray** | Click the lightning bolt icon |
| **Right-click Menu** | Tray icon → Open |

---

## 🕹️ Playing Games

### Quick Play

1. Press `Ctrl+Alt+G` to open the app
2. Click any game card
3. Play!

### Game Categories

- **All** - 41 games
- **Favorites** - Your pinned games
- **Arcade** - Fast-paced action (25 games)
- **Puzzle** - Brain teasers (10 games)
- **Classic** - Timeless games (3 games)
- **Strategy** - Strategy (1 game)
- **Creative** - Creative (2 games)

### Search

- Type in the search box to find games
- Real-time fuzzy matching
- Clear with the ✕ button

### Random Game

- Click the 🎲 button
- Discover new games

## ⚙️ Basic Settings

### Changing Theme

1. Click the Settings icon
2. Select the Display tab
3. Choose a theme
4. 11 themes available: Neon, Retro, Minimal, Forest, Ocean, Sunset, Purple, Matrix, Cyberpunk, Dark Blue, Fire

### Audio Settings

- Master Volume - Overall volume
- SFX Volume - Sound effects
- Music Volume - Background music
- Mute on Blur - Mute when window loses focus

### Difficulty Level

- **Easy** - For beginners
- **Normal** - For intermediate players
- **Hard** - For experienced players
- **Extreme** - For experts
